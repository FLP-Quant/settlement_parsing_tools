# Data Directory

This directory contains supporting data files and maps used by the project.

## Structure

- `maps/`: Contains mapping files
- `config/`: Configuration files
- `raw/`: Raw data files

## Data Format

Data files should be stored in a consistent format, preferably:
- CSV files for tabular data
- JSON files for structured data
- YAML files for configuration

## Adding New Data

When adding new data files:
1. Place them in the appropriate subdirectory
2. Update this README with a description of the new data
3. Document the data format and structure
4. Add any necessary data validation scripts
